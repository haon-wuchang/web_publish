== 상세페이지 == 
상세페이지 왼쪽 상품디테일컷 = 상품정보탭의 사진갯수와 동일한 경우도 있고 아닌경우도 있음.... 

상품이미지 호버 시 왼쪽의 상품이미지크기와 동일하게 오른쪽에 호버한 이미지가 나오게됨


색상옵션 있는상품과 없는상품 상세페이지 폼이 약간 다름
단색 : 배송방법 밑에 사이즈만 나옴(버튼으로 ) xs,s,m,l ...
색상옵션있는경우 : 배송방법 밑에 색상, 색상선택옵션, 사이즈선택옵션 이 있음
색상 버튼으로 선택 시 해당 색상의 상세페이지로 넘어감 ( 색상마다 상세페이지 각각 만듬 )


상품상세페이지에서 상세페이지 탭부분까지 내리면 
오른쪽에서 옵션선택창이 나옴 ( 사이즈, 배송, 수량 , 가격 , 장바구니, 바로구매 )
창 닫기 버튼 있어서 닫으면 오른쪽에 계속 구매하기 창이 떠있음


상세페이지 탭 : 상품정보, 사이즈&핏, 리뷰, 추천 총4개의 탭으로 구성 : 근데 안에 구성내용이 브랜드별로 조금씩 다름.;;
